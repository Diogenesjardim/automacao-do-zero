# 🔧 Git e GitHub

## 📥 Clonar repositório

No GitHub, clique em "Code" e copie a URL.

No terminal:

git clone URL_DO_REPOSITORIO

---

## 📂 Entrar na pasta

cd automacao-do-zero

---

## ➕ Adicionar arquivos

git add .

---

## 💾 Salvar alterações

git commit -m "meu primeiro commit"

---

## 🚀 Enviar para o GitHub

git push

---

## ⚠️ Possíveis erros

### Erro de autenticação
O GitHub não usa mais senha.

Você precisa usar um TOKEN.

👉 Como resolver:
- Vá em Settings no GitHub
- Developer settings
- Personal access tokens
- Generate new token

Use ele no lugar da senha

---

## 🧠 Dica

Sempre que alterar algo:

git add .
git commit -m "descrição"
git push
