# Java API

Projeto de testes de API em Java usando Rest Assured e JUnit.

## Requisitos

- Java JDK 17 ou superior
- Apache Maven

## Como executar

1. Abra um terminal na pasta `03-java-api`
2. Rode:
   ```bash
   mvn test
   ```

## O que está aqui

- `pom.xml` — dependências do projeto
- `TestApi.java` — classe de teste de API

## Observações

- O Maven baixa todas as dependências automaticamente.
- Certifique-se de ter `JAVA_HOME` configurado corretamente.
